Description

These examples use the reverse function to rearrange the elements of the Vector. The Vector, with its elements in reverse order is returned.

 

Details

The elements of the FltVector will be returned in reverse order.

(reverse #(float| 5.456699848175 3.141592025757 )) Returns: #(float| 3.141592025757 5.456699848175 )
