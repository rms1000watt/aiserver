Description

This example shows that the tail key-binding value of a Dictionary must first be defined before it can be accessed by the cdr function. If no tail value is defined, then the cdr function returns #void.

 

Details

(define ClassRec #{dic|Bldg: 200 Room: 34 Period: 1 } ) Returns: #<Dictionary 188141>

The tail value of the ClassRec Dictionary is defined.

(setCdr ClassRec #{dic| Teacher: "Smith" Subject: "Math" }) Returns: none
(display ClassRec) Returns: (#{dic| Bldg: 200 Room: 34 Period: 1 } . #{dic| Teacher: "Smith" Subject: "Math" })

Displays only the tail value of the ClassRec Dictionary.

(display (cdr ClassRec)) Returns: #{dic| Teacher: "Smith" Subject: "Math" }
