Description

These examples use the reverse function to rearrange the elements of the Vector. The Vector, with its elements in reverse order is returned.

 

Details

The elements of the ShortVector will be returned in reverse order.

(reverse #(short| 11 12 ) ) Returns: #<ShortVector 123456>
(display #<ShortVector123456>) Returns: #(short| 12 11 )
