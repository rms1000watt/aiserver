Description

In this example, the Virtual Machine Lambda will be created using the makeLambda function (alternatively, the compile function could be used to generate the Virtual Machine instructions).

 

Details

This is an Lambda to execute the following Lisp script:

(lambda (n) (+ n 10)) Returns: #<Lambda 123456 >

The same Lambda can be created via the following Lisp expression:

(makeLambda Av: #{n: 0} Pv: #{__C0 10}) Returns: #<Lambda 234567>

The same result can be obtained by the following Lisp expression:

(compile '(lambda (n) #void)) Returns: #<Lambda 345678>
