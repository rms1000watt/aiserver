Description

These examples use the compareGT function to compare two objects and return the Boolean value true if the first argument is greater than the second argument or false if otherwise. The Integer argument is converted into Character values.

 

Details

(compareGT #\a #\b ) Returns: false

The character, a, has an ASCII value of 97

(compareGT 97 #\a) Returns: false
