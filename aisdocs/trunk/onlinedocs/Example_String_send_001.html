Description

This example first defines the method EatSpace. It then invokes the addMethod function to add the method in the Methods Dictionary. The method EatSpace is invoked by the send function calling the message in the String Methods Dictionary.

 

Details

Define a method named EatSpace which will removes spaces from a String.

(defun EatSpace ( S )
vars: (NewString, i, j, len)
(setq NewString (new String: ""))
(setq j 0)
(setq len (length S))
(loop for i from 0 until len do
;;Move the character to the buffer only if it isn't a space
(if (<> S[i] #\space)
(begin
(setq NewString[j] S[i])
(setq j (add1 j ))
) ;; end begin
) ;; end if
) ;; end loop
NewString) ;; End EatSpace
Returns: modified String

Add the method EatSpace which responds to the message Eat: to the String Methods Dictionary.

(addMethod String: Eat: EatSpace ) Returns: #<Dictionary 12345>

Define a String that contains some spaces

(setq S (new String: "Hello World" )) Returns: Hello World

Invoke the method named Eat to remove spaces from a String

(setq Y (send Eat: S)) Returns: HelloWorld

The method can be invoked simply by name:

(setq Y (Eat: S)) Returns: HelloWorld
