Description

This example takes 2 existing Bit Vectors and appends the values of each Bit Vector. The result is placed in a new Bit Vector.

 

Details

(setq X (new Vector: bit: 5 1)) Returns: #<BitVector 123456>
(setq Y (new Vector: bit: 4 0)) Returns: #<BitVector 234567>
(append X Y ) Returns: #<BitVector 456789>
(display #<BitVector 456789> ) Returns: #(bit| 1 1 1 1 1 0 0 0 0 )
