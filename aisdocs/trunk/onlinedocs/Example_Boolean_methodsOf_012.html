Description

This example defines a method and a message named toEnglish that converts the Boolean Value, true, to the English word "Yes" and converts the Boolean value, false, to the English word "No". Then it accesses the defined mthod through the methodsOf function.

 

Details

Add a message named toEnglish to the Boolean Methods Dictionary.

(addMethod Boolean: toEnglish: (lambda(b) (if (= b 1) (begin 0) else (begin 1)))) Returns: #<Dictionary 123456>

Query the methods Dictionary for the Boolean Data Type:

(length (methodsOf Boolean: )) Returns: 1

Get the names of the methods in the methods Dictionary for the Boolean Data Type:

(ref (methodsOf Boolean:)[0 0]) Returns: toEnglish
(ref (methodsOf Boolean:)[0 1]) Returns: #<Lambda 123456>
(isMember (methodsOf Boolean: ) toEnglish ) Returns: true
