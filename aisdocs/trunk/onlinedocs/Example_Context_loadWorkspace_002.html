Description

This example saves and loads the workspace from a file using the file ID..

 

Details

(clear) Returns: true

Create and initialize a global variable.

(setq mtest "hello") Returns: hello

A file containing the workspace is opened.

(setq fid (fileOpen "workspace.db" 1 3)) Returns: 1

The current workspace is saved using the file ID.

(saveWorkspace fid) Returns: true
(fileClose fid 1) Returns: true

The current workspace is cleared.

(clear) Returns: true

There is no previously saved value for mtest.

(writeln mtest) Returns: #void

A file containing the workspace is opened.

(setq fid (fileOpen "workspace.db" 0 3)) Returns: 1

The current workspace is loaded using the file ID.

(loadWorkspace fid ) Returns: true

The saved value of mtestin the currently loaded workspace is displayed.

(writeln mtest) Returns: hello
