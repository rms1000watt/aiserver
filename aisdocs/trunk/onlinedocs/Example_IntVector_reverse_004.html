Description

These examples use the reverse function to rearrange the elements of the Vector. The Vector, with its elements in reverse order is returned.

 

Details

The elements of the IntVector will be returned in reverse order.

(reverse #(int| 11 12 ) ) Returns: #<IntVector 123456>
(display #<IntVector 123456>) Returns: #(int| 12 11 )
