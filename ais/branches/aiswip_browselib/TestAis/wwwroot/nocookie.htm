Analytic Information Server (AIS)
Cannot Set Cookie

AIS was unable to set a cookie on your browser.

AIS requires that that cookies are enabled on your browser.
If you are using Internet Explorer, check that cookies are enabled as follows:

    1. Select the tools menu item at the upper left corner of the browser window.
    2. Select Internet Options... from the drop-down menu list.
    3. Select the security tab in the Internet Options dialog window.
    4. Select Internet Zone icon (or Intranet Zone if a local connection).
    5. Click on the Custom Level... button.
    5. Scroll down in the Settings window to Cookies. Make sure that cookies are enabled.
    6. Select OK and then OK again to close the options dialog.
    7. You may need to restart your application.
