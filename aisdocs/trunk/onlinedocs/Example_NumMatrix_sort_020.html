Description

These examples show how the sort function sorts the elements of the NumMatrix in descending and ascending order.

 

Details

(setq X #(nummat| 9 "e" -4.954 9.432 3)) Returns: #<NumMatrix 123456>

Sort a NumMatrix in ascending order and returns a NumMatrix containing the sorted values.

(sort X < ) Returns: #<NumMatrix 123456>

Displays the sorted NumMatrix. The Text value is replaced with 0.0.

(display X ) Returns: #(nummat| -4.954 0.0 3.0 9.0 9.432 )
(setq X #(nummat| 9 #\A -4.954 9.432 3)) Returns: #<NumMatrix 234567>

Sort a NumMatrix in descending order and returns a NumMatrix containing the sorted values.

(sort X > ) Returns: #<NumMatrix 123456>

Displays the sorted NumMatrix. The Character is replaced with its Integer ASCII value of 65.

(display X ) Returns: #(nummat| 65.0 9.432 9.0 3.0 -4.954 )
