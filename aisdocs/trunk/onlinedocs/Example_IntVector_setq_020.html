Description

If the setq function specifies a value that does not exist in the specified Vector, the value will be dynamically installed.

 

Details

Sets the Vector.

(define X #(int| 11 12 ) ) Returns: #<IntVector 123456>
(setq X 4 2) Returns: #<IntVector 123456>

Displays the contents of the modified Vector.

(display X) Returns: #(int| 11 12 0 0 2 )

Notes and Hints

The set function keyword may be used instead of setq.
