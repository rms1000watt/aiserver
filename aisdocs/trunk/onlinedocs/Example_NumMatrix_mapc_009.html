Description

The following example will define a user function, makePlural, that will take an argument (which is a Matrix value). The mapc function then applies each Matrix value as argument to the function makePlural.

 

Details

A NumMatrix is defined.

(define Scores #(nummat| 5 8 4 9 ) ) Returns: #<NumMatrix 123456>

The result of the add1 operation of the last value is shown.

(setq NewScores (mapc add1 Scores)) Returns: 10

Notes and Hints

The mapc function does not produce a duplicate Matrix as in the map function.
