Description

If the setq function specifies a value that does not exist in the specified Matrix, the value will be dynamically installed. This example shows the setq function replacing the old value in the Vector with a new value.

 

Details

Sets the Matrix.

(define Fruits #(mat| "apple" "orange" "lemon")) Returns: #<Matrix123456>

Sets another Matrix.

(setq Food (new Matrix: 2 2 2 "apple" "orange" "lemon" "pear")) Returns: #<Matrix234567>

The newly set value "tomato" replaces the old value "apple".

(setq Fruits 0 "tomato") Returns: #<Matrix 123456>

Displays the contents of the modified Matrix.

Fruits[0] Returns: tomato

The newly set value "potato" replaces the old value "orange".

(setq Food 1 "potato") Returns: #<Matrix23456y>
(display Food) Returns: #(mat[2 2]| "apple" "potato" "lemon" "pear" )

The newly set value "meat" replaces the old value "lemon".

(setq Food 1 0 "meat") Returns: #<Matrix234567>
(display Food) Returns: #(mat[2 2]| "apple" "potato" "meat" "pear" )

Notes and Hints

The set function keyword may be used instead of setq.
