Description

This example converts all lowercase characters of the specified String into uppercase characters.

 

Details

Defines a String

(define S "Mary had a little lamb") Returns: Mary had a little lamb
(setq T (upcase S)) Returns: MARY HAD A LITTLE LAMB
(display S) Returns: MARY HAD A LITTLE LAMB
(display T) Returns: MARY HAD A LITTLE LAMB

Notes and Hints

The upcase function is destructive for String arguments. The original String argument is modified in place.
