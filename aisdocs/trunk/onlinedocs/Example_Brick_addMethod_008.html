Description

This example uses the added method addUtilities which will add a new bill to the sum owed for Utilities in a Brick.

 

Details

Define a method addUtilities which adds the amount in the Utilities field.

(addMethod Brick: addUtilities: (lambda (r bill) (+= r.Utilities bill))) Returns: #<Dictionary 123456>

Define a Brick that contains the amount of each bill for this month.

(define monthlyBills (new Brick: 20 Name:Object:1 Mortgage:Number:1 Utilities:Number:1 Visa:Number:1)) Returns: #<Brick 123456>

Sets the values of the Brick field.

(setq monthlyBills.Name #("John Doe")) Returns: #<Brick 123456>
(setq monthlyBills.Mortgage: 2100.0) Returns: #<Brick 123456>
(setq monthlyBills.Utilities: 100.0) Returns: #<Brick 123456>
(setq monthlyBills.Visa: 600.0) Returns: #<Brick 123456>

Invoke the method named addUtilities to compute the total amount checks for Utilities written for the month. The arguments to the function are the Brick name and a Number value.

(send addUtilities: monthlyBills 300.0) Returns: #<Brick 123456>

The method can be invoked simply by name:

(addUtilities: monthlyBills 300) Returns: #<Brick 123456>

Notes and Hints

To view the entry in the methods Dictionary created by the addMethod function, you can use the method name to access the directory entry by

#<Dictionary 123456>[methodName:] or

To determine the number of assigned methods:

(length (methodsOf Brick: ))

To access the nth method name:

(methodsOf Brick: )[n 0]

To view the Methods Dictionary:

(display (methodsOf Brick: ))
