Description

These examples use the compareLT function to compare two objects and return the Boolean value true if the first argument is lesser than the second argument or false if otherwise. The Integer argument is converted into Character values.

 

Details

(compareLT #\a #\b ) Returns: true

The character, a, has an ASCII value of 97

(compareLT 97 #\a) Returns: false
