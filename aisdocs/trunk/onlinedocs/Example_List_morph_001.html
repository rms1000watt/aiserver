Description

This example shows simple macro substitution using the morph function.

 

Details

(morph '((++ x))) Returns: #<Pair 123456>

The ++ is a built-in macro.

(display #<Pair 123456>) Returns: (setq x (+ x 1))
(morph 1) Returns: 1
(morph '(1)) Returns: 1
