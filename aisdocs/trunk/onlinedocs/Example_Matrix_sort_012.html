Description

These examples show how the sort function sorts the elements of the Matrix in descending and ascending order.

 

Details

Defines a Matrix of String values..

(setq X #(mat| "aardvark" "zebra" "lion" "cheetah")) Returns: #<Matrix 123456>

Sort a Matrix in descending order and returns a Matrix containing the sorted values.

(sort X > ) Returns: #<Matrix 123456>

Displays the sorted Matrix

(display X ) Returns: #(mat| "zebra" "lion" "cheetah" "aardvark" )

Sort a Matrix in ascending order and returns a Matrix containing the sorted values.

(sort X < ) Returns: #<Matrix 123456>

Displays the sorted Matrix

(display X ) Returns: #(mat| "aardvark" "cheetah" "lion" "zebra" )
