Description

This example shows that the AIS only has Pair and NOT List as the symbolic name for the Pair Data Type. However, Lists are considered also as Pairs because a List is really a collection of pairs chained together.

 

Details

(setq X '(employee name job salary ) ) Returns: #<Pair 197815>
(isType Pair: X) Returns: true

The container X contains a Pair because a List is really a collection of pairs chained together.

(isType List: X) Returns: false
