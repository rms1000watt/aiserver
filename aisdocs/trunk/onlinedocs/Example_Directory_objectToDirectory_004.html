Description

This example shows how the objectToDirectory updates the values of the {directory1} object with the values of the {directory2} object only if the key matches. If the key does not match the new key and its value are added to the {directory1} object.

 

Details

Defines the first directory object.

(setq a (new Directory: A: 4 B: 6)) Returns: #<Directory 123456>

Defines the second directory object.

(setq b (new Directory: B: 7 C: 8)) Returns: #<Directory 234567>

Updates the first directory object with the values of the second directory object.

(objectToDirectory a b) Returns: #<Directory 345678>

Display the contents of the updated Directory object.

(display #) Returns: #{ dir| A: 1 B: 7 C: 8}
