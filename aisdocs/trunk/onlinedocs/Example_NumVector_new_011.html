Description

These examples show how to construct a typed Vector. A Number Vectors is constructed using the new function.

 

Details

Returns a Number Vector of repeating 4's and 5's.

(new Vector: Number: 5 4 5) Returns: #<NumVector 234567>

Displays the contents of the Number Vector.

(display #<NumVector 234567>) Returns: #(num| 4.0 5.0 4.0 5.0 4.0 )
