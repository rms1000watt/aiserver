Description

The setq function assigns an Integer value to a previously uninitialized variable with the value #void.

 

Details

X is currently empty.

(ref x) Returns: #void

Shows the data type of x.

(type x) Returns: #void

Assigns an Integer value to the variable.

(setq x 1) Returns: 1
(type x) Returns: Integer

Notes and Hints

The set function keyword may be used instead of setq.
