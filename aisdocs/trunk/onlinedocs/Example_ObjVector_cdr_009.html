Description

This example shows how the the cdr function accesses the tail value of a Vector.

 

Details

(define ClassRec #(obj| #{A: 1 B: 2} #(1 2 ) .#{Teacher: "Smith" Subject: "Math"} )) Returns: #<ObjVector 188141>

The cdr value is assigned to the variable TeacherRec.

(setq TeacherRec (cdr ClassRec)) Returns: #<Structure 123456>
(display TeacherRec) Returns: #{Teacher: "Smith" Subject: "Math"}

Displays the value of the Vector with index 0.

ClassRec[0] Returns: #<Structure 188141>

Displays the value of the Vector with index 1.

ClassRec[1] Returns: #<Vector 188141>

The index 2 will not reach the structure that was defined as the cdr

ClassRec[2] Returns: #void

Notes and Hints

The cdr is not reachable by the [] operator.
