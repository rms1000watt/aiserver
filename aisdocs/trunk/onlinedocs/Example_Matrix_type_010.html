Description

This example shows how the type function return the user-defined data type of the AIS Objects evaluated.

 

Details

Define a number Matrix and check the type.

(setq X (new Matrix: number:1 2 11.433 -2.6)) Returns: #<NumMatrix 123456>
(type X) Returns: NumMatrix

Define a Matrix and check the type.

(setq X #(mat| "alpha" "beta" "gamma")) Returns: #<Matrix 123456>
(type X) Returns: Matrix

Define a Number Matrix and check the type.

(setq Z (new Matrix: number: 1 2 #{A: 1 B: 2} #(1 2) ) ) Returns: #<NumMatrix 123456>
(type Z) Returns: NumMatrix
