Description

These examples show the makeQuotedSymbol function converting a String into a Quoted Symbol object.

 

Details

The variable Z is assigned a QuotedSymbol object.

(setq Z (makeQuotedSymbol "My Symbol")) Returns: My Symbol
(type Z) Returns: QuotedSymbol

Here, a new String is created and then converted into a Quoted Symbol.

(setq X (makeQuotedSymbol (new String: "John"))) Returns: John
(type X ) Returns: QuotedSymbol
