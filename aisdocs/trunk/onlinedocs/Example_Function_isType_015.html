Description

The isType function is used to check if the argument object is of the type Function. If it is, the isType function returns true, otherwise, it returns false.

 

Details

(isType Function: muli) Returns: true
(isType Function: addi) Returns: true

The julian function returns a Number.

(isType Function: (lambda (a) (+ a 1))) Returns: true
