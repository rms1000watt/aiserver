Description

This example shows how the objectToDictionary transforms the {source} Vector into a Dictionary object.

 

Details

(define aVector #(Name: "Bill" SSN: "616 55 1209" Age: 45) ) Returns: #<Vector 123456>

Transforms the source Vector argument into a Dictionary.

(setq Dict (objectToDictionary aVector) ) Returns: #<Dictionary 123456>

Checks the type of the transformed Vector.

(type Dict) Returns: Dictionary

Displayed the contents of the Dictionary object.

(display Dict) Returns: #{dic|| Age 45 Name "Bill" SSN "616 55 1209"}true
