Description

These examples check if the datatype of the object is an alphabetic character. If it is, the function returns true, otherwise, it returns false.

 

Details

(isCharAlphabetic #\a) Returns: true
(isCharAlphabetic #\^) Returns: false
(isCharAlphabetic "Abcde") Returns: true
(isCharAlphabetic "Ab2de") Returns: false
(isCharAlphabetic #<Dictionary 47917>) Returns: false
