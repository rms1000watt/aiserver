Description

The setAttributes function returns the keys of the Dictionary and saves it in an Object Vector.

 

Details

Reset Attributes and Values

(setq X #void) Returns: none

Reset Attributes and Values

(setq Y #void) Returns: none

Create an unattributed Vector.

(setq X #("Camaro" $22590 #Feb,1,1996) ) Returns: #<Vector 123456 >

Create an unattributed Vector.

(setq Y #("Mustang" $26990 #Mar,1,1996) ) Returns: #<Vector 234567>

Create an attribute Vector.

(setq attb (new Vector: object: 3 Product: Amount: Date:) ) Returns: #<ObjVector 123456>

Create a block of Vectors.

(setq block (new Dictionary: a: X b: Y) ) Returns: #<Dictionary 123456>
(display block[0] ) Returns: #("Camaro" $22,590 #Feb,1,1996)
(display block[1] ) Returns: #("Mustang" $26,990 #Mar,1,1996)

The Vector contents of the Dictionary has no attribute values.

(display (refAttributes block[0])) Returns: #void
(display (refAttributes block[1])) Returns: #void

Attribute all Vectors in block.

(setAttributes block attb) Returns: #<Dictionary 123456>

The attribute of each Dictionary element.

(display (refAttributes block[0])) Returns: #(objvector | Product: Amount: Date:) true
(display (refAttributes block[1])) Returns: #(objvector | Product: Amount: Date:)
