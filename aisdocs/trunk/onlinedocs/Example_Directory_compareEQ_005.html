Description

These examples use the compareEQ function to compare two Directories and return the Boolean value, true or false. The keys and values of each Directory are compared to determine whether the Directories are equal or not.

 

Details

The keys and values of the two Directories are equal.

(compareEQ #{dir|X: 1 Y: 2} #{dir|X: 1 Y: 2}) Returns: true

Each key-value pair is compared with its corresponding key-value pair in the order given in Directory. In this example, the first value in dir1 is not equal to the first value in dir2, and so on.

(compareEQ #{dir|X: 1 Y: 2} #{dir|Y: 2 X: 1 }) Returns: false

The keys and values of the two Directories are not equal.

(compareEQ #{dir|A: 1 B: 2} #{dir|X: 1 Y: 2}) Returns: false
