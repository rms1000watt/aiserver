Description

The version function returns the current version identifier of the Analytic Information Server engine.

When to use

The version function is used to query the workspace the version of the Analytic Information Server engine.

 

Syntax

(version)


Name Description AIS Types
none" ---

Returns:

The Version of the Analytic Information Server engine.
