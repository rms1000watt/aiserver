Description

These examples show how the sort function sorts the elements of the Structure and returns an integer vector containing the sorted indices of either the Stucture key or value.

 

Details

Sort a Structure by values in descending order and returns a Vector.

(sort #{A: 2 B: 3 C: 1} > byValue: true) Returns: #<IntVector 123456>

The integer vector contains the indices of the Structure elements sorted by values.

(display #<IntVector 123456> ) Returns: #(int| 1 0 2 )

Sort a Structure by keys in ascending order and returns a Vector.

(sort #{C: 2 B: 1 A: 3} < true) Returns: #<IntVector 234567>

The integer vector contains the indices of the Structure elements sorted by keys.

(display #<IntVector 234567>) Returns: #(int| 2 1 0 )
