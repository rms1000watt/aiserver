Description

These examples check if the vecotr object contains only alphabetic characters. If it does, the function returns true, otherwise, it returns false.

 

Details

(isCharAlphabetic #\a) Returns: true
(isCharAlphabetic #\^) Returns: false
(isCharAlphabetic MySymbol:) Returns: true
(isCharAlphabetic 12345:) Returns: false
(isCharAlphabetic #<Dictionary 47917>) Returns: false
(isCharAlphabetic "Abcdefgh") Returns: true

The Object conains the number 2.

(isCharAlphabetic "Ab2cdefgh") Returns: false
