Introducing: Agent Information Server™

Agent Information Server™(AIS) is an analytic database designed to store massive amounts of heterogeneous data, upon which complex computer science algorithms are to be employed. The goal is to convert data, whose information content is a mystery, into data whose information content is understood.

AIS Guides

AIS provides a series of guides that are listed in the next section. These guides are divided into two categories. The first are end-user guides that explain how to use AIS. An example is the "AIS User's Guide" which describes the AIS IDE (Integrated Development Environment). The second category includes those guides that are intended for both end-users and for AIS developers. Examples that fall into this category are the "Help Documentation Guide" and the "Programmer's Style Guide". The first guide provides guidance for using the refGuide documentation facility for generating help documentation. The second example provides suggested coding practices for five of the programming languages that end-users and AIS developers will encounter.
