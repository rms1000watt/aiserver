Description

These examples check whether the object evaluated is a Number Matrix or not.

 

Details

The evaluated object is a NumMatrix, not a Pair, so true is returned.

(isAtom #(nummat| 3.2 9.0 23) ) Returns: true

The evaluated object is a Pair, so false is returned.

(isAtom '(1 2)) Returns: false
