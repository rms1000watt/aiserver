Description

These examples use the reverse function to rearrange the elements of the NumMatrix. The NumMatrix, with its elements in reverse order is returned.

 

Details

The elements of the NumMatrix will be returned in reverse order.

(reverse #(nummat| 1 2 3 4)) Returns: #<NumMatrix 123456>
(display #<NumMatrix 123456>) Returns: #(nummat| 4.0 3.0 2.0 1.0 )

The elements of the Number Matrix will be returned in reverse order.

(reverse #(nummat| 1.54 2 3.53 -4)) Returns: #<NumMatrix 123456>
(display #<NumMatrix 123456>) Returns: #(nummat| -4 3.53 2 1.54 )
