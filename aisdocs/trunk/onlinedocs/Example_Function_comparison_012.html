Description

These examples use the comparison functions to do a case-sensitive comparison of two objects of the same type. A Boolean value is returned.

 

Details

(> sub1 add1 ) Returns: true
(< sub1 add1 ) Returns: false
(= add1 add1 ) Returns: true

Notes and Hints

If you prefer an integer resulting from the comparison (0 for equal, -1 for less than, and 1 for greater than), use the compare function.
