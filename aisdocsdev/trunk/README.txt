
-------------------------------------
| How TO GENERATE AIS DOCUMENTATION |
-------------------------------------
1. Save your modified or added document/s in the CoreContent folder.
2. Clear the foundry folder.
3. Copy CoreContent, Guides, Templates to foundry.
4. Run RefGuide which generates result that goes into wwwroot folder.
5. Check in modification in wwwroot to aisdocs project. 
6. Click on AStartup.html in wwwroot or in the aisdocs project.  That should launch the AIS Homepage.

--------------------------------------------
| Create or update AIS Documentation Index |
---------------------------------------------
1. AisDocsDev should contain the following folders: Templates, Bin, wwwroot.
2. Run IndexRefGuide which generates the index for the documents in CoreContent.
3. The files generated by IndexRefGuide are saved in the Bin Folder.
4. Run AliceStartup.sl in the top-level of AisDocsDev. 
5. Click the "View Application" icon on the upper left side of webide.  It should launch the homepage.
