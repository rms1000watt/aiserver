Description

This examples show how a Vector value is referenced by using its index. The ref function is used either with the Vector Name or the Vector's Object ID.

 

Details

Define a Vector containing various fruit names:

(define Fruits #("apple" "orange" "lemon")) Returns: #<Vector 123456>

Accesses the Vector value with index 0.

(ref Fruits 0) Returns: apple

Accesses the Vector value with index 1.

(ref Fruits 1) Returns: orange

Accesses the Vector value with index 2.

(ref Fruits 2) Returns: lemon

Instead of the Vector name, the object ID may be used to access the Vector values

(ref #<Vector 123456> 0) Returns: apple

Accesses the Vector value with index 1.

(ref #<Vector 123456> 1) Returns: orange

Accesses the Vector value with index 2.

(ref #<Vector 123456> 2) Returns: lemon
