Description

These examples show how to construct a typed Vector. A Bit, Number, Pcode and Word Vectors are constructed using the new function.

 

Details

Returns a Bit Vector of all 1's.

(new Vector: Bit: 5 1) Returns: #<BitVector 123456>

Displays the contents of the Bit Vector.

(display #<BitVector 123456>) Returns: #(bit| 1 1 1 1 1 )

Returns a Number Vector of repeating 4's and 5's.

(new Vector: Number: 5 4 5) Returns: #<NumVector 234567>

Displays the contents of the Number Vector.

(display #<NumVector 234567>) Returns: #(num| 4.0 5.0 4.0 5.0 4.0 )

Returns an empty pcode vector.

(new Vector: Pcode: 0) Returns: <PcodeVector 345678>

Displays the contents of the Pcode Vector.

(display #<PcodeVector 345678>) Returns: #(pcd| )

Same as (new Vector: 0)

(new Vector: Word: 0) Returns: #<Vector 456789>

Returns an empty Vector.

(display #<Vector 456789>) Returns: #()
