Description

The char Function converts an ASCII numeric value into a text character. You can specify an argument using in the range 1 to 255.

When to use

Use the char to convert a numeric to an ASCII character.

 

Syntax

(char num)

Name Description AIS Types
num A integer from 1 to 255Integer

Returns:

The character equivalent of the integer argument
