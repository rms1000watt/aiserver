Description

This example shows how the type function return the data type of the AIS Objects evaluated.

 

Details

These examples show the different formats of the Date Data Type.

(type #Feb,14,1976 ) Returns: Date
(type (date 1994 2 3)) Returns: Date

The Julian date format is a numeric data type.

(type 729032.2455) Returns: Number

The Julian numeric format is converted into the Date data type.

(type (date 729032.2455)) Returns: Date
