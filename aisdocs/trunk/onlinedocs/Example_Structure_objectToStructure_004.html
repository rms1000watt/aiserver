Description

This example shows how the objectToStructure integrates all the key-value bindings of the source Dictionary objects.

 

Details

Defines the first Dictionary.

(setq a (new Dictionary: A: 4 B: 6)) Returns: #<Dictionary 123456>

Defines the second Dictionary.

(setq b (new Dictionary: B: 7 C: 8)) Returns: #<Dictionary 234567>
(objectToStructure a b) Returns: #<Structure 123456>

The values of A: B: and C: are copied in to the target Structure with the value of B: updated with the value of the second Dictionary argument.

(display #<Structure 123456> ) Returns: #{A: 1 B: 7}true
