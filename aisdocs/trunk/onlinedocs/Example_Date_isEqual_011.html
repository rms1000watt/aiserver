Description

This example use the julian function to get the numeric equivalent of the date argument.

 

Details

(julian #Jan,1,1996) Returns: 729023.0

The julian numeric equivalent is equal to the normal date format argument.

(isEqual 729023.0 #Jan,1,1996) Returns: true
