Description

These examples use the compare functions to do a case-sensitive comparison of two objects of the same type. An Integer value is returned.

 

Details

(define nameList (new Brick: 2 Name:Object:3)) Returns: #<Brick 123456>
(define customerList (new Brick: 2 Name:Object:1)) Returns: #<Brick 234567>

The first Brick has greater value than the second Brick.

(compare nameList customerList) Returns: 1
