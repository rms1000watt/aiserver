Description

This examples show how the Date value is returned using the ref function and using the shorthand method.

 

Details

(setq x #Jan,3,1954) Returns: #Jan,3,1954
(ref x) Returns: #Jan,3,1954

The shorthand form of getting the value of a variable is simply the variable name itself:

x Returns: #Jan,3,1954
